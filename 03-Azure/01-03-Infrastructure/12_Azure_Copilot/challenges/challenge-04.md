# Challenge 4 - Optimization Agent

**[Home](../Readme.md)** - [Previous Challenge](challenge-03.md) - [Next Challenge](challenge-05.md)

## Goal

The Optimization Agent helps you **reduce costs and carbon emissions** for your Azure resources while maintaining performance. It can:

- Surface **Azure Advisor cost recommendations** for your subscriptions
- Provide **detailed explanations** for each recommendation
- Offer **alternative optimization options** for comparison
- Generate **charts** showing expected results of applying recommendations
- Create **Azure CLI or PowerShell scripts** to implement optimizations
- Summarize **total potential savings** across your environment

Use the Optimization Agent in Azure Copilot to discover cost-saving opportunities, understand recommendations, compare alternatives, and generate scripts to implement optimizations.

**Scenario:** As Contoso Ltd.'s FinOps engineer, your manager has asked you to review the company's Azure spending and identify opportunities to reduce costs without impacting performance. The company has several VMs and Virtual Machine Scale Sets running across multiple subscriptions.

By the end of this challenge, you will be able to:

- Query cost-saving opportunities using the Optimization Agent
- Understand detailed recommendation explanations
- Request and compare alternative optimization options
- Read optimization charts generated by Azure Copilot
- Generate scripts (CLI/PowerShell) to apply optimizations
- Summarize potential savings across your environment

## Actions

### Pre-Challenge Setup

> **Note:** The workshop deployment scripts have already created an intentionally oversized VM for you. If you ran `lab/Deploy-Lab.ps1`, everything below is ready to use.

#### Workshop Resources (Pre-Deployed)

Resources in **`rg-copilot-<suffix>-ch03`** (in your chosen deployment region):

| Resource        | Name                   | SKU                                 | Purpose                                                   |
| --------------- | ---------------------- | ----------------------------------- | --------------------------------------------------------- |
| Virtual Machine | `vm-copilot-oversized` | Standard_D4s_v3 (4 vCPU, 16 GB RAM) | Deliberately oversized VM to trigger cost recommendations |

> **Why D4s_v3?** This SKU is intentionally too large for the idle workload it runs. Within 24-48 hours Azure Advisor will generate rightsizing recommendations that the Optimization Agent can surface.

#### Option A: Use the Pre-Deployed VM (Recommended)

1. Find your **subscription ID** in the Azure portal (navigate to **Subscriptions** and copy the GUID)
2. The VM's **ARM Resource URI**: `/subscriptions/{YOUR_SUBSCRIPTION_ID}/resourceGroups/rg-copilot-<suffix>-ch03/providers/Microsoft.Compute/virtualMachines/vm-copilot-oversized`
3. Proceed to Task 1

> **Note:** Azure Advisor recommendations may take **24-48 hours** to appear after the VM is deployed. If you don't see recommendations yet, try the prompts using your subscription ID — the agent will still show the workflow.

#### Option B: Use Your Own Existing VMs

If you already have VMs or VMSS in your subscription:

1. Note your **subscription ID** (GUID)
2. Note the **ARM Resource URI** of a VM (found in the resource's Properties page)
3. Proceed to Task 1

### Task 1: Discover Cost-Saving Opportunities (10 min)

1. Open Azure Copilot and **enable agent mode**
2. Ask for a subscription-wide overview:

   > _"Show me my top Azure Advisor cost recommendations for this subscription."_

3. Review the list of recommendations. For each one, note:
   - The resource affected
   - The recommended action
   - The estimated monthly savings
4. Try a broader query:

   > _"Summarize total potential monthly savings from all active Azure Advisor cost recommendations, grouped by category (VM rightsizing, reserved instances, idle resources)."_

**Question to answer:** What categories of cost savings does the Optimization Agent identify? How significant are the potential savings?

### Task 2: Deep-Dive into a Specific Recommendation (10 min)

Pick one recommendation from Task 1 and explore it in detail:

1. Ask for more details:

   > _"Give me detailed rightsizing analysis for VM `vm-copilot-oversized` in resource group `rg-copilot-<suffix>-ch03`: current SKU, recommended SKU, average CPU and memory utilization, estimated monthly savings, and performance impact."_

2. Understand the reasoning:
   - What metrics indicate the resource is underutilized?
   - What is the current vs. recommended SKU?
   - What is the expected performance impact?

3. Request an alternative:

   > _"Explain why Azure Advisor recommends resizing `vm-copilot-oversized` from its current SKU to the smaller SKU. Include the utilization data and the threshold rules Advisor uses."_

4. Compare the options:
   - Original recommendation vs. alternative
   - Cost savings vs. performance trade-offs

**Question to answer:** Does the agent provide enough detail to make an informed decision? How does it help you weigh cost vs. performance?

### Task 3: Visualize Optimization Impact (10 min)

1. Ask Azure Copilot to show you the expected impact visually:

   > _"Generate a bar chart comparing the current monthly cost and the recommended monthly cost for `vm-copilot-oversized` after applying the Advisor rightsizing recommendation, and show the % savings."_

2. Review the generated chart:
   - What metrics are shown (CPU, memory, cost)?
   - How does current usage compare to the recommended tier?
3. Ask for a subscription-level summary:

   > _"Give me a subscription-level optimization summary: total potential monthly savings, breakdown by resource type (VMs, storage, databases), and the top 3 recommendations by impact."_

**Question to answer:** How do the visual charts help you communicate optimization plans to management or stakeholders?

### Task 4: Generate Optimization Scripts (10 min)

Generate scripts to implement the recommended changes:

1. Ask for a PowerShell script:

   > _"Generate a PowerShell script to resize VM `vm-copilot-oversized` in resource group `rg-copilot-<suffix>-ch03` from Standard_D4s_v3 to Standard_D2s_v3. Include stop, resize, and start steps."_

2. Review the generated script:
   - Does it resize the VM?
   - Does it include a shutdown/restart step?
   - Does it handle any prerequisites?

3. Ask for a CLI version:

   > _"Now generate the equivalent Azure CLI (bash) script to resize `vm-copilot-oversized` in `rg-copilot-<suffix>-ch03` from Standard_D4s_v3 to Standard_D2s_v3 (deallocate, resize, start)."_

4. Compare the two scripts — note the differences in syntax and approach

5. **Important:** Do NOT run the scripts yet. First:
   - Review for accuracy
   - Verify the target resource
   - Consider the impact on running workloads (downtime during resize)

**Question to answer:** Are the generated scripts ready to execute, or do they need modification?

### Task 5: Explore Optimization from Different Entry Points (5 min)

Try accessing optimization recommendations through different methods:

1. **From the Azure Advisor page:**
   - Navigate to **Azure Advisor** → **Cost** recommendations
   - Find a VM recommendation
   - Click **"Optimize"** next to the resource
   - See how this launches the Copilot optimization conversation

2. **From Operations Center (if available):**
   - Navigate to **Operations Center** (preview)
   - Find a recommendation and click **"Optimize with Copilot"**

3. **Using specific resource URIs:**
   > _"Show me cost saving recommendations for `/subscriptions/{YOUR_SUBSCRIPTION_ID}/resourcegroups/rg-copilot-<suffix>-ch03/providers/microsoft.compute/virtualmachines/vm-copilot-oversized`"_

**Question to answer:** Which entry point is most convenient for your daily workflow?

## Success criteria

- You queried top cost-saving opportunities for a subscription
- You explored a specific recommendation in detail with explanations
- You requested and compared an alternative recommendation
- You reviewed a visualization/chart of optimization impact
- You generated at least one script (PowerShell or CLI) for optimization
- You understand multiple entry points for accessing optimization recommendations

## Learning resources

- The Optimization Agent surfaces **Azure Advisor cost recommendations** enhanced with AI-powered explanations
- It can **compare alternatives** so you can make informed decisions based on cost vs. performance trade-offs
- **Visual charts** make it easy to communicate optimization plans to stakeholders
- Generated **scripts** (CLI/PowerShell) accelerate implementation but should always be reviewed before execution
- The agent works from **multiple entry points**: Copilot chat, Azure Advisor, and Operations Center
- [Optimization Agent documentation](https://learn.microsoft.com/en-us/azure/copilot/optimization-agent)

**Limitations to Note:**

- Currently supports **VMs and VMSS** for optimization recommendations
- Azure Copilot **generates scripts but cannot execute them** — you must run scripts yourself
- Cannot automatically **create or update budgets and alerts**
- Shows a maximum of **10 items per query** (default is 5)
- Specify **subscription GUIDs or ARM Resource URIs** for best results (not just resource names)
