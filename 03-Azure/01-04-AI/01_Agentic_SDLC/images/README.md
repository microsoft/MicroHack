# Images

Store screenshots and diagrams for the **Agentic SDLC MicroHack** in this folder and
reference them from the challenge and walkthrough Markdown with relative links, e.g.:

```markdown
![Architecture](../images/architecture.png)
```

The high-level Agentic SDLC lifecycle is currently rendered as a Mermaid diagram
directly in [`../Readme.md`](../Readme.md). Add a rendered PNG here (for example an
architecture diagram of the Octocat Supply deployment on Azure, or challenge
screenshots) as the MicroHack grows.
