# Azure AI MicroHacks

A collection of MicroHacks focused on AI on Azure and the GitHub + Azure agentic platform.

## Available MicroHacks

* [**Agentic SDLC with GitHub Copilot & Azure**](01_Agentic_SDLC/Readme.md) — Take a full-stack application through the complete Agentic Software Development Lifecycle (plan → implement → test → review → deploy → operate → maintain) using GitHub Copilot, GitHub Actions, the Azure SRE Agent, and GitHub Agentic Workflows.
* [**Inventory Planning with Microsoft Fabric (low-code)**](02_Inventory_Planning_Fabric/README.md) — Build a closed sense → reason → act inventory-planning loop with **Microsoft Foundry Agent Service** prompt agents (no application code) grounded on a governed **Fabric Data Agent** doing NL2SQL over an inventory Lakehouse, with a human-in-the-loop approval gate.
