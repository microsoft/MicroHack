# Setting Up Azure NetApp Files 

## Objective:
Teach participants how to set up and configure Azure NetApp Files. 

## Topics Covered: 

* Creating NetApp accounts 

* Setting up capacity pools 

* Creating and managing volumes 

## Activities: 

* Demo: Live demonstration of setting up Azure NetApp Files. 

* Hands-On Lab: Participants create their own NetApp accounts, capacity pools, and volumes.
