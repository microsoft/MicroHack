# Managing and Monitoring Azure NetApp Files 

 

## Objective: 
Equip participants with the skills to manage and monitor Azure NetApp Files. 

## Topics Covered: 

* Monitoring performance and health 

* Managing Active Directory integration 

* Backup and restore operations 

## Activities: 

* Demo: Monitoring and management tools in the Azure portal. 

* Hands-On Lab: Simulate common management tasks like setting quotas and performing backups.
